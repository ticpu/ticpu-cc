# ticpu-cc

Personal Claude Code skill marketplace.

## Install

```
claude plugin marketplace add ticpu/ticpu-cc
claude plugin install freeswitch@ticpu-cc
claude plugin install bcachefs@ticpu-cc
```

## Plugins

### freeswitch

Skills:

- **freeswitch:usage** — Dialplan, variables, ESL, time routing, basic SIP
- **freeswitch:dev** — Core media C code, codec implementations, CI/CD
- **freeswitch:sofia** — mod_sofia SIP internals, flags, constants, RTP, NAT

### bcachefs

- **bcachefs:bcachefs** — bcachefs filesystem internals

## Update

```
claude plugin marketplace update ticpu-cc
claude plugin update freeswitch@ticpu-cc
claude plugin update bcachefs@ticpu-cc
```
